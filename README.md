# TKU TIMDA for WRS 2020

## Requirements

### Setup:
```bash
$ mkdir -p wrs_ws/src && cd wrs_ws
$ git clone https://github.com/tku-iarc/wrs2020.git src/
```

### ROS Packages:
```bash
$ sudo apt-get install ros-<distro>-pcl-ros
$ sudo apt-get install ros-<distro>-camera-info-manager
```
